Этот симулятор естественного отбора генерирует ситуацию, в которой разные существа борются за выживание. Поле поделено на несколько точек с едой, каждую из которых случайным образом занимают до 2 существ. Если существо одно, оно набирает достаточно еды, чтобы выжить и размножиться. Каждое существо может создать всего одного потомка за один этап. Если животных несколько, они начинают конкурировать за еду, исходя из их уровня агрессии. Агрессивные животные понижают отбирает и уничтожает запасы еды у другого. Максимальный уровень агрессии предполагает, что два животных, которые им обладают, при встрече уничтожат друг друга. Если в результате столкновения у существа не достаточно еды для выживания, его уровень агрессии повышается, а в противном случае понижается.
